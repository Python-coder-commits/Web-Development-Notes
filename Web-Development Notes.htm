Lesson 1: What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create webpages and web applications. HTML describes the structure of a web page using markup.

<html>
    <head>
        <title>My First Web Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>
  • HTML is easy to learn and widely used.
  • Every webpage begins with a <html> tag.
  • The <body> tag contains the visible content.

Practical:

Create an HTML file with the above structure and open it in a browser to see the output.

Expected Output:

A webpage with the title "My First Web Page" and a heading "Hello, World!".

Lesson 7: Forms in HTML

HTML forms are used to collect user inputs, such as text, email, and numbers. Here's an example of how to use a simple form in HTML.

<Html>
<body> 
<form action="submit-form.php" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">

    <label for="email">Email:</label>
    <input type="email" id="email" name="email">

    <label for="age">Age:</label>
    <input type="number" id="age" name="age">

    <button type="submit">Submit</button>
</form>
</body>
</Html>
  • Use <form> to define a form.
  • Use <input> fields to get user input.
  • Submit the form with a <button>.

Practical:

Create a form that collects your name, email, and age, then submit it.

Expected Output:

A form that collects user details and submits them.

Lesson 8: Input Fields in HTML

Input fields are used to capture user data:

<html>
<body> 
<div>
        <label for="name">Name:</label>
        <input type="text" id="name" placeholder="Enter your name"><br><br>
                    
        <label for="email">Email:</label>
        <input type="email" id="email" placeholder="Enter your email"><br><br>
                    
        <label for="password">Password:</label>
        <input type="password" id="password" placeholder="Enter your password"><br><br>
                    
        <button onclick="alert('Submitted!')">Submit</button>
</div>
</body>
</html>
  • <label>: A label associated with an input field.
  • <input>: The element used to capture user input.
  • type: Specifies the type of input field (e.g., text, email, password).
  • placeholder: Provides a hint to the user about what to enter.
  • <button>: Creates a clickable button.
  • onclick: Adds functionality to the button, such as displaying an alert message.

Practical:

Create a layout with input fields for a user's name, email, and password, along with a button that displays an alert when clicked.

Expected Output:

A webpage with fields for name, email, and password, and a button that shows an alert when clicked.

Lesson 10: Horizontal Rule in HTML

The horizontal rule, created using the <hr> tag, is used to create a thematic break or a visual divider in your content:

<html>
<body> 
    <h1>Welcome to Lesson 10</h1>
                    
    <p>This is the first section of text.</p>
    <hr>
    
    <p>This is the second section of text, separated by a horizontal rule.</p>
    <hr style="border: 2px dashed blue;">
                    
    <p>This is the third section of text, with a styled horizontal rule above it.</p>
    </body>
    </html>
  • <hr>: Inserts a horizontal line to divide sections of content.
  • style: Adds custom styling to the horizontal rule (e.g., color, width, or border style).

Practical:

Create a webpage with three sections of text separated by horizontal rules. Add custom styling to at least one of the horizontal rules.

Expected Output:

A webpage with three sections of text divided by horizontal lines, one of which is styled differently.

Lesson 11: Buttons in HTML

Buttons are an essential part of interactive web elements. In HTML, buttons can be created using the <button> tag. You can use buttons in various ways, including for submitting forms or creating clickable actions:

<html>
<body>
    <button>Click Me!</button>
                        
    <button type="submit">Submit</button>
    
    <button type="reset">Reset</button>
    
    <a href="https://www.example.com">
    <button>Go to Example</button>
</a>
</body>
</html>
  • <button>: Defines a clickable button.
  • type="submit": Specifies a submit button within a form.
  • type="reset": Defines a button to reset form fields.
  • <a>: Wraps the button to make it behave like a link.

Practical:

Create a webpage with three different buttons: one normal button, one submit button, and one reset button. Test their functionality and see how they work.

Expected Output:

A webpage with three buttons: one normal, one submit, and one reset. Clicking the submit button will simulate a form submission, and the reset button will reset form fields if included in a form.
